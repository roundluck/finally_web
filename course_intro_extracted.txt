Understand the significance of enterprise application development.
Appreciate the fundamentals of Enterprise Architecture and development frameworks.
Learn how to develop, deploy and test enterprise applications

Deliverables:

Live demonstration to the class (5-10 mins) 
 only for individual assignment.

The following are required to be submitted:
Functional document 
 listing all the user functions developed in the application
Technical document
 describing the architecture of the application, class diagrams, entity-relationship diagrams and any other technical aspects that the student wishes to describe.
Database dump
Source code

All the above-described deliverables should be uploaded to a discussion thread in NTU Learn website.
